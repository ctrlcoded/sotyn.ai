# sotyn.ai

Official marketing website for **sotyn.ai**, an AI-powered ERP platform built for **MEPF, Solar, Civil, and EPC contractors** across India.

Built with **Astro**, the website is optimized for speed, SEO, and lead generation while providing a modern SaaS experience.

## Live Website

**https://sotyn.ai**

---

## Features

- Modern responsive UI
- Built with Astro for maximum performance
- SEO optimized
- AI-search friendly
- Demo booking system
- WhatsApp lead integration
- Customer login redirect
- Fast static site generation
- Mobile-first design
- JSON-LD structured data
- Sitemap & Robots.txt

---

## Getting Started

### Clone the repository

```bash
git clone https://github.com/ctrlcoded/sotyn.ai.git
cd sotyn.ai
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open:

```
http://localhost:4321
```

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```

## Lead Generation

The demo booking form submits customer inquiries through the configured webhook. If the webhook is unavailable, it automatically redirects users to WhatsApp with a pre-filled message to ensure no leads are missed.

The website is currently live at:

Upload the generated `dist/` directory to any static hosting provider, including:

- Vercel
- Netlify
- Cloudflare Pages
